解决分支冲突
===========

## 知识点

* git merge [branch_name]

## 实战演习

~~~bash
$ git branch
$ git checkout master
$ git merge dev
$ nano index.htm
...
$ git add index.htm
$ git commit -m "merged by leader."
$ git log
$ git branch
~~~
