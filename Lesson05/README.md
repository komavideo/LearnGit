查看提交履历
===========

## 知识点

* git log命令的使用

## 实战演习

~~~bash
$ git log
$ nano index.htm
...
...
$ git add index.htm
$ git commit -m "added code."
$ git log
$ git log --oneline
$ git log -p
$ git log --stat
$ git log --help
~~~
